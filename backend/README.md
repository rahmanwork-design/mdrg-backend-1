# MDRG Backend Server

## Quick Start

### Windows
```
Double-click: start-server.bat
```

### Mac/Linux
```bash
./start-server.sh
```

### Manual
```bash
npm install
npm start
```

## Server URL
- API: http://localhost:3001/api
- Health Check: http://localhost:3001/api/health

## Database
Location: `database/mdrg.db`

Open with DB Browser for SQLite to view/edit customer data.
